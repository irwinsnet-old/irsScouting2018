Server Running

    Server setup

    1. Run server.bat in irsScouting2017
    2. Open Scout Director at "localhost:8080" in browser
    3. Setup usb connection

    Match not set

    1. Check Director table to find tablet that is not waiting
    2. Consult scouter for completion
    3. If match must be set manually, click current match to go to match select page
    4. Enter proper match in box and click set match

    Tablet not connecting

    1. Make sure most other tablets connected
    2. If most not working, check server and hub cable
    3. If single tablet, make sure it is tethered, replace cable if needed
    4. If tethered, open command promt by right clicking on start menu button
    5. Run command "ipconfig"
    6. Scroll up to find ethernet ipv4 address of tablet
    7. Click Tablets on Scout Director page for list of existing ips
    8. Make sure tablet ip matches config ip not on the server list
    9. Wait for up to a minute

Data Output

    Output Tableau Page

    1. Open Tableau on laptop
    2. Go to main with password "irs1318" if needed
    3. Go to relevant page at bottom left
    4. Click on worksheet at top right
    5. Select export/image and then save, save, and yes

    Match data output

    1. Connect and tether tablet
    2. Output Six teams Tableau page (Check match number)
    3. Open Scout Viewer on tablet at "{tablet ip}:8080/view"
    4. On tablet, select six team under images
    5. Unplug and go

    Full Data output

    1. Connect USB
    2. Go to Scout Viewer and select Output data
    3. Download file and move to USB
    4. Output Fuel, Gears, and Climbing pages in Tableau
    5. On usb go to images shortcut, or go to irsScouting2017/Server/web/images
    6. Copy Fuel.png, Gears.png, and Climbing.png to USB
    7. Unplug and go
